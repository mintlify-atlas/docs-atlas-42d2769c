## What is Doss Cashier App?

Doss Cashier App is a React Native mobile application for cashiers working on the Doss platform. It provides everything a cashier needs to:

- **Start and end shifts** — clock in and out with a live timer displayed on the home screen
- **Request payments** — search for customers by ID and send payment requests for exact amounts
- **Monitor transactions** — view real-time transaction status with success, refund, and cancel counts
- **Manage history** — browse paginated transaction history grouped by date, with the ability to refund or cancel
- **Receive notifications** — get foreground and background push alerts via Firebase and Notifee
- **Work offline** — the app detects connectivity changes and queues retries automatically

The app communicates with the Doss backend at `https://dossapp.com/api/` using Bearer token authentication and React Query for data fetching and caching.

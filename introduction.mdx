Doss Cashier App is a React Native mobile application built for cashiers on the Doss platform. It provides a focused set of tools for managing shifts, processing payment requests, reviewing transaction history, and staying up to date with real-time notifications — all from a single mobile interface on Android and iOS.

## Project structure

```
CashierApp/
├── src/
│   ├── assets/          # Images, fonts, and other static assets
│   ├── components/      # Shared UI components
│   ├── navigation/      # React Navigation stack and tab definitions
│   ├── screens/         # Feature screens (Home, Auth, Notifications, etc.)
│   ├── services/        # API clients, Firebase, notification handlers
│   ├── store/           # Zustand state stores
│   ├── themes/          # Colors, typography, toast styles
│   ├── types/           # TypeScript type definitions
│   └── utils/           # Utility functions
├── android/             # Android native project
├── ios/                 # iOS native project (Xcode workspace + CocoaPods)
├── index.js             # App entry point
└── App.jsx              # Root component
```
